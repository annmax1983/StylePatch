# StylePatch
[English](../README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | 日本語 | [Français](README_fr.md)

軽量なブラウザ拡張機能で、任意のウェブページの背景色、文字色、フォントサイズを即座にカスタマイズ。

> Chromium ベース · Manifest V3 · ゼロ追跡 · サイトごと設定

---

## 機能

| 機能 | 説明 |
|------|------|
| 🎨 **背景色・文字色** | ネイティブカラーピッカーまたはHEXコード直接入力 |
| 🔠 **フォントサイズスケーリング** | 80%〜150%の範囲で調整 |
| 👁️ **プリセットテーマ** | ライト、ウォーム、グリーン、ダーク — ワンクリック適用 |
| 🔄 **グローバルトグル** | 設定を失わずにグローバルに有効/無効 |
| 🚫 **サイトブラックリスト** | 特定サイトをスタイリングから除外 |
| 💾 **サイトごと設定** | サイトごとに異なるスタイルを保存、再訪問時に自動復元 |
| ⚡ **リアルタイムプレビュー** | すべての変更が即座に適用、ページリロード不要 |
| 🔒 **最小権限** | `storage` + `host_permissions` のみ |
| 🏗️ **Manifest V3** | `chrome.scripting.insertCSS`使用、コンテンツスクリプトゼロ |

## インストール

1. ブラウザの拡張機能ページを開く：`chrome://extensions/`
2. **デベロッパーモード**を有効にする
3. **パッケージ化されていない拡張機能を読み込む**をクリック
4. ツールバーのStylePatchアイコンをクリックして開始

---

## 使い方

1. ツールバーのStylePatchアイコンをクリック
2. カラーピッカーで色を選択、またはHEXコードを入力
3. プリセットを選択 — ライト、ウォーム、グリーン、ダーク
4. フォントサイズをスライダーで調整（80%〜150%）
5. **適用して保存**をクリック
6. ↺で元の外観に戻す
7. 「このサイトを除外」でブラックリストに追加

---

## プライバシー

- `storage` + `host_permissions`権限のみ
- 追跡なし、分析なし、外部データ送信なし
- すべてのデータはブラウザにローカル保存
