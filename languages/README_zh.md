# FontFixer

[English](../README.md) | 中文 | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md) | [Français](README_fr.md)

一款轻量级浏览器扩展，优化网页字体，提升阅读体验。一键更改字体、字号和文字颜色。

> Chromium 内核 · Manifest V3 · 最小权限 · 完全本地

---

## 为什么选择 FontFixer？

很多网站字体模糊、难看或太小。FontFixer 让你用喜欢的字体覆盖网页字体，调整字号，自定义文字/链接颜色 — 全部实时生效，无需刷新页面。

| 优势 | 说明 |
|------|------|
| 🔤 **本地字体支持** | 通过 `queryLocalFonts` API 读取设备上安装的字体 |
| ⚡ **实时预览** | 所有修改即时生效，无需刷新页面 |
| 💾 **按站点记忆** | 不同网站保存不同字体设置 |
| 🌐 **全局模式** | 所有网站统一应用一套字体配置 |
| 🔒 **最小权限** | 仅 `storage` + `scripting` + `activeTab` |

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 🔤 **字体选择** | 3 套内置字体（Noto Sans、思源黑体、Arial）或设备上任意字体 |
| 📏 **字号缩放** | 80% ~ 160% 滑块调节 |
| 🎨 **文字颜色** | 自定义文字颜色拾色器 |
| 🔗 **链接颜色** | 独立链接颜色，阅读更清晰 |
| 🔄 **范围控制** | 仅当前网站 或 所有网站 |
| 💾 **自动保存** | 设置按站点自动保存 |
| ↺ **一键重置** | 恢复网站原始字体 |
| 🌍 **6 种语言** | 英文、中文、日文、西班牙文、德文、法文 |

## 安装方法

1. 打开浏览器扩展页面：
   - **Chrome**：`chrome://extensions/`
   - **Edge**：`edge://extensions/`
2. 开启右上角**开发者模式**
3. 点击**加载已解压的扩展程序**，选择 `font-fixer` 文件夹
4. 点击工具栏中的 🔤 FontFixer 图标即可开始使用

---

## 使用方法

1. 点击工具栏中的 FontFixer 图标
2. 从下拉框选择字体（或点击内置字体标签）
3. 用滑块调整字号
4. 可选：修改文字和链接颜色
5. 点击**应用** — 即时生效

---

## 隐私保护

- `storage` — 本地保存字体偏好，不存储网页内容
- `scripting` — 注入 CSS 修改页面字体，不读取页面文字或数据
- `activeTab` — 仅在点击应用时访问当前标签页
- 无追踪、无分析、无外部连接
