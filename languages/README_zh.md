# StylePatch
[English](../README.md) | 中文 | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md) | [Français](README_fr.md)

一款轻量级浏览器扩展，可即时自定义任何网页的背景色、文字色和字号。

> 基于 Chromium · Manifest V3 · 零追踪 · 按站点保存设置

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 🎨 **背景色与文字色** | 通过原生取色器选取或直接输入十六进制色值 |
| 🔠 **字号缩放** | 80% 至 150% 无级调节 |
| 👁️ **预设主题** | 亮色、暖色、绿色、暗色 — 一键切换 |
| 🔄 **全局开关** | 全局启用/禁用扩展，设置不丢失 |
| 🚫 **站点黑名单** | 排除特定网站不应用样式 |
| 💾 **按站点保存** | 不同网站保存不同样式，再次访问自动恢复 |
| ⚡ **实时预览** | 所有修改即时生效，无需刷新页面 |
| 🔒 **最小权限** | 仅需 `storage` + `host_permissions` |
| 🏗️ **Manifest V3** | 使用 `chrome.scripting.insertCSS`，零内容脚本开销 |

## 安装

1. 打开浏览器扩展页面：
   - **Chrome**：`chrome://extensions/`
   - **Edge**：`edge://extensions/`
2. 开启**开发者模式**（右上角开关）
3. 点击**加载已解压的扩展程序**，选择项目文件夹
4. 点击工具栏中的 StylePatch 图标即可开始

---

## 使用方法

1. **点击工具栏中的 StylePatch 图标**
2. **选取颜色** — 使用原生取色器或输入十六进制色值
3. **选择预设** — 亮色、暖色、绿色或暗色
4. **调整字号** — 拖动滑块，80% 至 150%
5. **保存** — 点击 **应用并保存**，设置将持久化到当前站点
6. **重置** — 点击 ↺ 恢复网站原始外观
7. **排除** — 点击"排除此站点"加入黑名单
8. **切换** — 使用开/关按钮，禁用但不丢失设置

点击应用后设置自动保存，再次访问同一网站时自动恢复。

## 隐私

- 仅需 `storage` + `host_permissions` 权限
- `storage`：在本地保存自定义颜色、字号、站点黑名单和主题偏好，不存储网页内容
- `host_permissions`：仅用于注入自定义 CSS 样式，不读取 DOM 文本、图片、表单数据或 Cookie
- 不访问浏览历史，不追踪用户，不传输外部数据
- 所有数据保存在浏览器本地
- [隐私政策](https://annmax1983.github.io/StylePatch/privacy-policy.html)
